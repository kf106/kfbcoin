Allow subclients to join

This demonstrates that a client can admit more clients (i.e. if the master is down)

  1. Download the code from https://github.com/kf106/multichain-semi-open/archive/master.zip
  2. Extract the zip file into a folder, for example ~/multichain-semi-open/
  3. Install the Python virtual environment, MultiChain binaries and other requirements with the command:
        sudo ./install.sh
  4. Then follow the instructions below

Running the client

If you are joining from this node and running a new client for the first time, start it with the following command and parameters:

    ./run-client.sh {{ nodeaddress }} PLACEHOLDERsignup

If you have run the client node before, your blockchain node should be present, so just start it with the command:

    ./run-client.sh
