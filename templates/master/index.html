Welcome to KFBcoin!

The system currently only runs on Linux, and has been tested on Ubuntu 18.10.

  1. Download the code from https://github.com/kf106/kfbcoin/archive/master.zip
  2. Extract the zip file into a folder, for example ~/kfbcoin/
  3. Install the Python virtual environment, MultiChain binaries and other requirements with the command:
        sudo ./install.sh
  4. Then follow the instructions below

When you sign up you will receive one free kidney bean coin, and one free fava bean coin!

Running the client

If you are running the client node for the first time, start it with the following command and parameters:

    ./run-client.sh {{ nodeaddress }} PLACEHOLDERsignup

If you have run the client node before, your blockchain node should be present, so just start it with the command:

    ./run-client.sh
